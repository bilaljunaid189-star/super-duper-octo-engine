How to deploy (easy, recommended): GitHub Pages (web UI)
1. Go to https://github.com and log in.
2. Click the "+" then "New repository".
3. Repository name: bilaljunaid189-star.github.io
4. Set Visibility: Public
5. Click "Create repository".
6. In the new repo page click "Add file" → "Upload files".
7. Drag & drop `index.html` and `style.css` (from this package).
8. Commit changes.
9. Wait a minute — your site will be live at: `https://bilaljunaid189-star.github.io`

Alternate quick option: Netlify (no GitHub required)
1. Go to https://app.netlify.com/ and sign up (free).
2. In Dashboard click "Add new site" → "Deploy manually".
3. Drag & drop the `index.html` and `style.css` files.
4. Netlify will give you a live URL immediately.
